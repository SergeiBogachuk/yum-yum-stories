# IronDrop Dumpster Trailers

Standalone static landing page for a US dumpster trailer / junk removal business.

## Files

- `index.html` - page structure and content
- `styles.css` - visual design, layout, responsive behavior
- `script.js` - mobile menu, scroll reveal animation, mailto quote form
- `assets/` - custom SVG trailer visuals, including the owner-operated single-trailer illustration

## Run locally

Open `index.html` directly in a browser, or serve the folder locally:

```bash
cd /Users/sergeibogachuk/Documents/codex/dumpster-trailer-site
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
